# 地域/可用区名称变更公告
尊敬的用户：

为了解决优刻得在产品地域可用区名称命名方面存在的不规范问题，优刻得将对地域可用区名称进行规范化治理，旨在提升产品可用区名称的标准化程度，确保名称的清晰性和一致性，从而方便用户在使用产品时能够更直观地理解和选择对应的地域可用区。

地域可用区是指云计算服务提供商在不同地理位置设立的数据中心，以便为用户提供更高效和稳定的服务。不同的地域可用区可能会因其地理位置、基础设施、网络延迟等因素而影响用户的使用体验。因此，合理的命名不仅有助于用户快速识别和选择合适的服务区域，也有助于优化资源的配置和管理。

在此背景下，优刻得将对现有的地域可用区名称进行全面审查和评估，识别出名称中存在的模糊、重复或不符合行业标准的部分。这一过程将包括与行业专家的合作，参考国际通用的命名规范，并结合用户的反馈，以确保新命名方案的合理性和可操作性。

以下场景地域/可用区名称将按照治理后的规范名称展示，包括：
- 产品购买页
- 订单支付页
- 财务中心下页面
- 资源统计
- 官网页面
